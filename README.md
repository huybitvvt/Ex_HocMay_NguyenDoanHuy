# Bài tập Optimization với CNN trên CIFAR-10

Thư mục này gồm notebook Colab để làm bài thực hành Optimization cho bài toán phân loại ảnh CIFAR-10. Notebook đã có sẵn phần train các mô hình, so sánh kết quả và xuất báo cáo PDF.

## File chính

- `Bai_tap_Optimization_CIFAR10.ipynb`: notebook chạy trên Google Colab.
- `cifar10_optimization.py`: bản script Python cùng logic với notebook.
